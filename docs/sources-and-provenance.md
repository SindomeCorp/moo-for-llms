# Sources And Provenance

example, or dataset row should be traceable to either original authorship or a
compatible public source.

## Accepted Source Categories

- `original`: written for this repository and released under MIT.
- `manual-summary`: original summary based on a linked manual.
- `curated-public`: adapted from a public, compatible source with attribution.
- `approved-generic-sindome`: generic Sindome-derived code reviewed for public
  release, used with permission from Sindome (https://www.sindome.org/), and
  stripped of game-specific implementation details.

## Citation Policy

- Prefer links and short summaries.
- Do not paste long manual sections into this repo.
- If an example is adapted from existing code, name the source and license.
- Strip trailing VMS note/version/last-modified comments from Sindome-derived
  verb examples; those comments are commit metadata.
- When uncertain about provenance, mark the item as unreviewed and do not include
  it in release datasets.
